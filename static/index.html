Hack Kosice 2022 application

Welcome to Hack Kosice registration portal!

We are really excited that you chose to participate in Hack Kosice 2022. We hope that the application process will be as smooth as possible and we will see you on the hackathon. However the number of hackers we can take is only 150 so make sure you showcase you best qualities on the application to get chosen.

Important info about the application process.

  • Set aside around 15-30 minutes to completely fill out the application.
  • There are multiple sections in the application. You can use them to save your progress and return later
  • If you want to join the hackathon as part of the team, firstly fill out the application individually and afterwards you can create and join teams in your dashboard
  • We will review application on regular basis and update the status as well as send you an email about our decision
  • Start the application process by clicking continue and creating you account in MyMlh which you can use to sign-in into our registration portal
