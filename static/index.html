Hack Kosice 2022 Application

Welcome to Hack Kosice Application Portal!

We are really excited that you chose to participate in Hack Kosice 2022. We hope that the application process will be as smooth as possible and we will see you on the hackathon. However the number of hackers we can take is only 150 so make sure you showcase you best qualities on the application to get chosen.

Important info about the application process.

  • Set aside around 15-30 minutes to completely fill out the application
  • There are multiple sections in the application. You can use them to save your progress and then return later
  • You may want to join the hackathon as part of the team. First, fill out the application individually and afterwards you can create and join teams in your dashboard
  • We will be reviewing applications on regular basis and we'll update you on the status of your application, as well as send you an email regarding our decision
  • You can start the application process by clicking the apply now button and signing in with your MyMLH account, which you can use to access the registration portal
